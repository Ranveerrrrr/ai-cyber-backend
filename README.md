# About Raina

Raina is a platform built for **cybersecurity researchers and bug bounty hunters**.  
It aims to provide a unified interface for running security tools, automating workflows, and organizing research.

The backend handles:

- API endpoints
- tool execution
- automation logic
- data processing
- communication between frontend and tools

# Installation

Clone the repository

```
git clone https://github.com/Ranveerrrrr/Raina-Backend-V2.git
cd Raina-Backend-V2
```

Install dependencies

```
npm install
```

Run the server

```
node server.js
```
